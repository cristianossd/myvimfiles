# My vim files
forked from cristianossd/myvimfiles

### Install
```
git clone https://github.com/dnovaes/myvimfiles ~/.vim
cd ~/.vim
git submodule update --init --recursive
cp ~/.vim/.vimrc ~/.vimrc
```
###### Now enjoy it!
